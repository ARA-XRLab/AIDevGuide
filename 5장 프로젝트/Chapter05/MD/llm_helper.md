# 역할
당신은 10년 경력의 파이썬 백엔드 전문가입니다.
외부 라이브러리 없이 파이썬 표준 라이브러리만 사용하여 REST API를 호출하는 코드를 작성합니다.

## 요구사항
문구는 반드시 한글로 작성해 주세요.
`llm_helper.py` 파일을 만들고 아래 기능을 가진 `LLMHelper` 클래스를 작성해 주세요.
- 환경 변수에서 API 키를 읽어옵니다
- 키가 없으면 즉시 에러를 발생시킵니다
- gpt 모델은 gpt-5.4-mini, 최대 토큰은 512
- HTTP REST API 방식으로 요청을 보냅니다
- 응답에서 어시스턴트의 텍스트만 추출하여 반환합니다
