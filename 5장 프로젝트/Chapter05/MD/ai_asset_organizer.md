# 역할
당신은 10년 경력의 파이썬 백엔드 전문가입니다.
언리얼 엔진 에디터 스크립트 환경에서 동작하는 파이썬 코드를 작성합니다.

## 배경
이 코드는 언리얼 엔진 에디터에서 선택한 에셋들을 LLM에 전달하여
적절한 폴더 경로와 이름을 추천받는 자동화 스크립트입니다.
LLM 호출은 별도 모듈(`llm_helper.py`)의 `LLMHelper` 클래스를 사용합니다.

## 요구사항
아래 구성 요소를 가진 `ai_asset_organizer.py` 파일을 작성해 주세요.
문구는 반드시 한글로 작성
1. **시스템 프롬프트**
   LLM에게 전달할 역할과 지시를 담은 `SYSTEM_PROMPT` 문자열 상수를 만듭니다
   - LLM의 역할: 언리얼 엔진 테크니컬 아티스트
   - 아래 언리얼 에셋 접두사 규칙을 포함합니다
     - SM_: Static Mesh
     - SK_: Skeletal Mesh
     - T_: Texture
     - M_: Material
     - MI_: Material Instance
   - LLM이 반드시 JSON 형식으로만 응답하도록 출력 형식을 지정합니다

2. 세부기능
   - 언리얼 에디터에서 현재 선택된 에셋 목록을 가져옵니다
   - LLM에 에셋 목록을 전달하고 결과를 받아옵니다
   - `LLMHelper`를 이용해 에셋 목록을 전달하고 결과를 받아와 출력합니다.
