ASpaceRabbitGameMode에 점수 시스템을 추가하고 점수가 증가할 때 로그도 출력해줘. Coin의 OnCoinCollected 델리게이트가 호출되면 점수를 획득하게 해줘. UI에서 바인드해서 사용할 OnScoreChanged 델리게이트도 선언해줘.

요구사항:
- int32 CurrentScore 변수
- AddScore(int32 Amount) 함수
- GetCurrentScore() 함수
- 점수 획득 시 UE_LOG로 ‘Score : 1’ 과 같은 형식으로 로그 출력
- OnScoreChanged 델리게이트 (UI 바인딩용)
- Coin의 OnCoinCollected이 호출되면 점수 획득
- 게임오버 상태에서는 점수 추가 불가
- Epic Games 코딩 표준 적용
