점수를 표시하는 UMG 위젯인 USpaceRabbitHUD를 생성해줘. USpaceRabbitHUD을 기반으로 하는 위젯블루프린트 “WBP_HUD”는 내가 언리얼 에디터에서 제작할꺼야. GameMode의 BeginPlay에서 위젯을 생성하고 뷰포트에 추가하는 코드도 작성해줘. UI상에 ‘Score : ‘을 따로 표시할 꺼고 ScoreText에는 점수 값만 반영할꺼야. 

요구사항:
- UMG 사용
- 점수 표시용 UTextBlock (ScoreText)
- UpdateScore(int32 NewScore) 함수
- NativeConstruct에서 GameMode의 OnScoreChanged에 바인딩
- GameMode의 BeginPlay에서 위젯을 생성하고 뷰포트에 추가
- Epic Games 코딩 표준 적용
