게임오버 시 표시되는 UMG 위젯인 UGameOverWidget 클래스를 생성해줘. 언리얼 엔진 에디터에서 위젯 블루프린트는 직접 제작할꺼야. 팝업 형태의 UI이고 UI의 중앙 상단에는 ”GAME OVER”라는 타이틀이 있고 그 아래 최종 점수를 표시할 꺼야. UI상에 ‘FinalScore : ‘을 따로 표시할 꺼고 FinalScoreText 에는 점수 값만 반영해줘. 그리고 하단에는 Restart버튼과 Quit종료 버튼이 존재해. Restart버튼을 누르면 현재 레벨을 다시 로드하도록 구현하고 Quit 버튼을 누르면 종료되도록 구현해줘. 게임모드에서 게임이 시작할 때 입력모드를 GameOnly 및 마우스 커서 비활성 하고, 게임오버가 되면 입력모드를 UIOnly 및 마우스 커서 활성화 해줘. 

요구사항:
- UMG 사용
- 최종 점수 표시용 텍스트 (FinalScoreText)
- Restart 버튼 생성 및 재시작 처리
- Quit 버튼 생성 및 종료 처리
- SetFinalScore(int32 Score) 함수
- GameMode의 SetGameOver 함수에서 GameOverWidget을 생성하고 SetFinalScore 함수를 호출, 인풋모드를 UIOnly 및 마우스 커서 활성
- GameMode의 BeginPlay 함수에서 인풋모드를 GameOnly 및 마우스 커서 비활성
- Epic Games 코딩 표준 적용
