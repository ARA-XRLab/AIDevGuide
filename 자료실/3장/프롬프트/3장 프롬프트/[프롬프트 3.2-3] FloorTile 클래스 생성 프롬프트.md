AGENTS.md의 바닥 발판(FloorTile) 명세에 따라 AFloorTile 클래스를 생성해줘. FloorTile의 크기는 (1, 5, 1)로 해줘. 화면 왼쪽 끝을 벗어나면 OnTileExited 델리게이트를 호출하게 해서 FloorTile을 생성한 쪽에서 발판을 처리하도록 할꺼야. 이동할 때 Sweep은 사용하지 말고 구현해줘.

요구사항:
- UBoxComponent로 충돌 처리(플레이어가 밟을 수 있게), Box Extent의 크기는 (50, 250, 50)
- UStaticMeshComponent로 시각화 (프로토타입용 Cube)
- 발판의 가로 크기는 500cm
- 매 Tick에서 MoveSpeed만큼 왼쪽(-Y)으로 이동
- 화면 왼쪽 끝(DestroyY)은 -1500
- 화면 왼쪽 끝을 벗어나면 OnTileExited 델리게이트 브로드캐스트
- Epic Games 코딩 표준 적용
