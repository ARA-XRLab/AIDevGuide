AGENTS.md의 플레이어 명세에 따라 ASpaceRabbitCharacter 클래스를 생성해줘. 카메라는 내가 직접 레벨에 배치해 놨어. 카메라의 회전값은 (0, 0, 0)이고 위치값은 (-1000, 0, 0)이야. 주인공은 오른쪽 방향을 바라보며 제자리 점프를 할 수 있고 발판이 오른쪽에서 왼쪽으로 이동하는 방식이야. Enhanced Input System으로 스페이스바를 입력하면 점프하게 하고 최대 2단 점프를 구현해줘. 입력을 코드로 구현하더라도 입력 관련 에셋을 생성해서 블루프린트에서 적용할 수 있게 해줘. 중력(Gravity)은 직접 구현하고 LineTrace로 바닥 착지를 판정해서 바닥에 닿으면 다시 2단 점프를 뛸 수 있게 ApplyGravity() 함수와 Jump() 함수를 구현해줘. 

요구사항:
- UCapsuleComponent로 충돌 처리 (루트 컴포넌트)
- 스페이스바 입력 시 점프 (Enhanced Input System)
- 2단 점프 가능 (최대 2회)
- 중력 적용 (직접 구현, CharacterMovement 미사용)
- 바닥 착지 판정 (LineTrace)
- Epic Games 코딩 표준 적용
