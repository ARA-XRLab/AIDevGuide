AGENTS.md의 로켓(Rocket) 명세에 따라 ARocket 클래스를 생성해줘. 로켓은 자신의 앞 방향으로 직선으로 이동하게 해줘. 화면 왼쪽 밖으로 나가면 자동으로 파괴해줘. 발판에 생성되는 장애물과는 다르게 주인공의 점프를 방해하는 요소야. 주인공과 부딪히면 로켓이 파괴되면서 게임오버 되도록 처리해줘. 

요구사항:
- UBoxComponent로 충돌 감지 (Overlap 이벤트 사용)
- UStaticMeshComponent로 시각화 (프로토타입용 Cube, 주황색)
- 화면 오른쪽에서 왼쪽으로 직선 이동
- RocketSpeed 변수로 이동 속도 조절
- 플레이어와 충돌 시 게임오버 되도록 처리
- 화면 왼쪽 밖으로 나가면 자동 Destroy
- Epic Games 코딩 표준 적용
