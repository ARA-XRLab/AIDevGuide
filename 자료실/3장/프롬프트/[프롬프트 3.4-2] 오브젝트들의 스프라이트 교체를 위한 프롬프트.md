AFloorTile, AObstacle, ACoin, ARocket의 외형을 PaperSprite로 교체하고 싶어. 충돌체는 그대로 유지할꺼야. PaperSprite에 적용할 에셋은 내가 직접 언리얼 에디터에서 적용할꺼야. 그리고 AObstacle의 BoxExtent의 크기는 35로하고 발판에 생성 되었을 때 코인과 비슷한 높이에 배치되도록 해줘.

요구사항:
- 기존 MeshComponent를 UPaperSpriteComponent로 변경
- 각 스프라이트 에셋은 언리얼 에디터에서 직접 적용.
- BoxComponent(충돌)는 그대로 유지
- Epic Games 코딩 표준 적용
