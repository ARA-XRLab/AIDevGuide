AGENTS.md의 FloorManager 명세에 따라 AFloorManager 클래스를 생성하고 단계별로 코드를 작성해줘. 발판은 블루 프린트로 만들어진 것을 사용할꺼야. 처음에는 발판을 파괴하는 Y좌표인 -1500부터 오른쪽으로 발판 6개를 화면에 꽉 차게 이어지게 배치해서 주인공이 바닥으로 떨어지지 않게 해줘. 1초마다 발판을 생성하는데 발판의 재배치 위치는 가장 마지막으로 만든 발판의 현재 위치를 기준으로 재배치해줘. 그리고 발판은 화면에 최대 10개까지만 유지되게 해줘. 발판이 만들어 질 때 35% 확률로 발판의 간격을 만들어서 구멍을 만들어줘. 구멍의 크기는 발판의 절반인 250cm로 해줘. 구멍이 만들어진 졌다면 두번의 재배치는 구멍을 만들지 않게 해서 구멍이 연속적으로 생성되지 않게 해줘. 

단계별 요구사항:
1. BeginPlay에서 초기 발판 6개를 Y좌표가 -1500부터 오른쪽으로 생성 및 배치
2. 총 발판의 풀을 10개로 해서 발판이 모자라지 않게 생성
3. FloorTileClass는 블루 프린트로 만들어진 것을 사용
4. 각 FloorTile의 OnTileExited 델리게이트에 함수 바인딩
5. 1초마다 새로운 발판을 생성하고 생성 위치는 가장 마지막으로 만들어진 발판의 현재 위치를 기준으로 오른쪽에 재배치
6. 재배치 할 때는 평소에는 발판이 서로 붙어서 이어지고, 랜덤하게 35%확률로 250cm의 간격이 생김
7. 구멍이 연속으로 만들어지지 않게 구멍이 생긴 후 두번의 재배치는 구멍이 생기지 않게 처리
