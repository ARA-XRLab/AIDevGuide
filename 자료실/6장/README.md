### 02_Prompt — 실습 프롬프트 및 AI 응답

본문에 등장하는 프롬프트와 AI 에이전트별 응답 결과입니다.

```
02_Prompt/
└── [프롬프트 6.2-1] AGENTS.md 생성/
    ├── [프롬프트 6.2-1] AGENTS.md 생성.md   ← 프롬프트 원문
    └── Answer/
        ├── Codex.md      ← 본문 기준
        ├── Claude.md     ← 참고용
        └── Gemini.md     ← 참고용
```

본문은 Codex 응답 기준으로 작성되었습니다. Claude, Gemini 응답은 동일한 프롬프트에 대해 다른 AI 에이전트가 어떻게 응답하는지 비교 참고용으로 제공합니다.

> AI 에이전트가 직접 코드를 수정한 프롬프트는 Answer 폴더 대신 README.txt가 포함되어 있습니다. 실행 결과는 01_Project 소스 코드에 반영되어 있습니다.

## 📖 활용 방법

1. 실습 진행 중 막히면 `01_Project` 해당 절 폴더에서 완료 시점 프로젝트 확인
2. 프롬프트 결과가 본문과 다르면 `02_Prompt` 해당 폴더의 `Answer/Codex.md` 와 비교
3. 로그·프로파일링 데이터가 필요하면 `03_자료실` 해당 절 폴더에서 다운로드
