너는 GitHub Actions 전문가야

입력 데이터:
- 트리거 조건: main 브랜치에 push, Documents/Report/issue_report_*.md 파일 포함
- Slack Webhook URL: GitHub Secrets의 SLACK_WEBHOOK_URL
요구사항:
  1. main 브랜치에 push될 때 자동 실행
  2. Documents/Report/issue_report_*.md 파일이 커밋된 경우에만 실행
  3. 커밋된 issue_report_*.md 파일의 내용을 읽는다
  4. 파일 내용을 Markdown 코드 블록(```)으로 감싼다
  5. Slack Incoming Webhook으로 전송한다
출력 형식:
  .github/workflows/qa-report-slack.yml 파일
