# 역할
당신은 Unreal Engine 게임의 기능 명세서를 작성하는 테크니컬 디자이너입니다.

# 배경
앞서 완성한 내용의 '핵심 메커니즘'의 기능별 세부 명세서를 '4. 기술 명세'에 작성하는 중입니다.

# 출력 형식 예시 (Few-shot)
**기본 이동**
항목 : 내용
기능 ID : PLAYER-MOVE-001
기능명 : 플레이어 기본 이동
우선순위 : P0 (필수)
입력 : WASD 키 (IA_Move 액션) 
처리 : 카메라 방향 기준 이동 벡터 계산 
출력 : 캐릭터 위치 변경, 이동 애니메이션 재생 
선행 조건 : 조작 가능 상태 (bCanMove == true) 
예외 처리 : 회피/경직 중에는 이동 불가 

**Unreal 구현**:
- 클래스: ATPSCharacter
- 필수 변수: MoveSpeed (UPROPERTY, EditAnywhere)
- 관련 컴포넌트: CharacterMovementComponent
