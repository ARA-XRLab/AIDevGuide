# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
플레이어 애니메이션 인스턴스에 공격 애니메이션 재생 기능을 추가하고 사격 시 호출한다.

# 요구사항
- TPSPlayerAnimInstance에 변수명 AttackMontage를 추가하고 에디터에서 연결할 수 있도록 노출한다.
- PlayAttackAnim() 기능을 추가한다.
  - 블루프린트에서도 호출할 수 있도록 설정한다.
  - AttackMontage가 연결된 경우에만 재생한다.
  - 이미 재생 중이면 중복 재생하지 않는다.
- 플레이어 캐릭터의 사격 기능에서
  - 기존 발사 판정은 그대로 유지한다.
  - 애니메이션 인스턴스를 TPSPlayerAnimInstance로 변환한다.
  - 변환에 성공하면 PlayAttackAnim()을 호출한다.

# 제한
- 기존 발사 판정/데미지 로직은 수정하지 않는다.
- 애니메이션 그래프 관련 코드는 작성하지 않는다.
- 애니메이션 애셋을 코드에서 직접 불러오지 않는다.
