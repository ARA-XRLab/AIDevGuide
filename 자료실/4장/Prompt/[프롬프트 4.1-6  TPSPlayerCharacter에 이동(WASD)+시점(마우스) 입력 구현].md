# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
플레이어 캐릭터에 이동과 시점 회전을 구현한다.

# 입력 변수
- DefaultMappingContext(IMC_PlayerDefault)
- MoveAction(IA_PlayerMove)
- LookAction(IA_PlayerLook)

# 요구사항
- 게임 시작 시 DefaultMappingContext를 등록한다.
- 변수명 Move로 이동 기능을 구현한다.
- 입력 값 X는 앞뒤(ForwardDirection), 입력 값 Y는 좌우(RightDirection) 이동에 사용한다.
- 카메라가 바라보는 방향(Yaw)을 기준으로 이동한다.
- 변수명 Look 으로 시점 회전 기능을 구현한다.
- 마우스 좌우로 시점 좌우 회전, 마우스 상하로 시점 상하 회전을 처리한다.
- 캐릭터가 이동하는 방향을 바라보도록 자동 회전을 켠다.
- Enhanced Input 모듈을 프로젝트 의존성에 추가한다.

# 제한
- 점프 입력은 추가하지 않는다.
- 기존 코드는 필요한 부분만 최소한으로 수정한다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
