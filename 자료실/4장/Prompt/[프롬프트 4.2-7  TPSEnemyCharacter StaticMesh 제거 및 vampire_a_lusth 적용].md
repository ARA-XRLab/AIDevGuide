# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
적 캐릭터의 임시 외형을 스켈레탈 메시로 교체한다.

# 요구사항
- 기존 StaticMesh 컴포넌트와 관련 설정을 헤더와 생성자에서 완전히 제거한다.
- 사망 연출에 사용하던 회전/오프셋/스케일/자동 제거 시간 변수도 제거한다.
- HandleDead( )에서 StaticMesh를 이용한 사망 연출 코드를 제거하고, 나중에 사망 애니메이션으로 교체할 예정이라는 메모를 남긴다.
- 자동 제거 시간은 3.0f로 고정한다.
- 코드에서 vampire_a_lusth를 직접 불러와 기본 메시 슬롯에 적용한다.
  - 로드 경로: /Game/Enemy/Model/vampire_a_lusth
- 메시의 위치와 회전 보정 값은 에디터에서 조정할 수 있도록 노출한다.
- 몸통(캡슐)은 사격 판정에서 제외하고, 메시 기준으로 사격 판정을 처리한다.

# 제한
- FSM/공격/피격 로직은 수정하지 않는다.
- 사망 애니메이션 실제 재생은 구현하지 않는다.
