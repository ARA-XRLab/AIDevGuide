# 역할
언리얼 엔진 5 C+ + 개발자, AGENTS.md 규칙을 따른다.

# 작업
적 AI에 사망 상태와 즉시 사망 처리, 임시 사망 연출을 구현한다.

# 요구사항
- 데미지를 받으면 즉시 사망 처리한다.
- 적 캐릭터에 HandleDead() 기능을 다음 순서대로 구현한다.
   1. FSM 컴포넌트의 NotifyDead()를 호출한다.
   2. 캐릭터 이동을 비활성화한다.
   3. 몸통(캡슐)과 외형 메시의 충돌을 모두 끈다.
   4. 외형 메시에 사망 연출을 적용한다.
      - 회전 값, 오프셋 값, 스케일 값을 에디터에서 조정할 수 있도록 노출한다.
   5. 일정 시간 후 자동으로 제거되도록 설정한다. 제거 시간은 에디터에서 조정할 수 있도록 노출한다.
- FSM 컴포넌트에 NotifyDead()를 다음과 같이 구현한다.
  - 중복으로 사망 처리되지 않도록 막는다.
  - 사망(Dead) 상태로 전환한다.
  - 사망 상태 진입 시 매 프레임 업데이트를 비활성화한다.
  - 사망 상태에서는 상태 업데이트를 즉시 건너뛴다.

# 제한
- HP 시스템은 구현하지 않는다.
- 래그돌 및 애니메이션은 구현하지 않는다.
- 기존 대기/이동/공격/피격 로직은 필요한 부분만 최소한으로 수정한다.
