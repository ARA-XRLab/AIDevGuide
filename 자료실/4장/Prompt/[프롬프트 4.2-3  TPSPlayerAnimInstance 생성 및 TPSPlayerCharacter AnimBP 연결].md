# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
플레이어 전용 애니메이션 인스턴스 클래스를 만들고 플레이어 캐릭터에 연결한다.

# 요구사항
- 클래스 이름: TPSPlayerAnimInstance
- 언리얼 엔진 기본 애니메이션 인스턴스를 상속받아 만든다.
- 다음 변수를 에디터에서 확인할 수 있도록 노출한다.
  - Speed(float): 앞뒤 이동 속도
  - Direction(float): 좌우 이동 속도
  - bIsInAir(bool): 공중 여부
- 매 프레임마다 다음 값을 갱신한다.
  - 소유 캐릭터를 TPSPlayerCharacter로 변환해서 가져온다.
  - Speed: 이동 방향과 캐릭터 앞 방향을 내적하여 계산한다.
  - Direction: 이동 방향과 캐릭터 오른쪽 방향을 내적하여 계산한다.
  - bIsInAir: 공중에 떠 있는지 여부를 계산한다.
- 플레이어 캐릭터 생성자에서 ABP_TPSPlayer_C를 코드로 불러와 메시에 연결한다.
  - 로드 경로: /Game/Animation/ABP_TPSPlayer

# 제한
- ABP_TPSPlayer, BS_PlayerMove 파일은 에디터에서 직접 만들고 코드로 생성하지 않는다.
- 기존 이동/카메라/사격 코드는 수정하지 않는다.
