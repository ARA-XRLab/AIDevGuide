# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
적 AI의 상태 머신을 컴포넌트로 구현한다.

# 요구사항
- 컴포넌트 이름: TPSEnemyFSMComponent
- 언리얼 엔진 기본 액터 컴포넌트를 상속받아 만든다.
- 상태 목록을 다음과 같이 정의한다.
    - None/Idle/Move/Attack/Hit/Dead
- 대기(Idle) 상태
    - 감지 범위 안에 플레이어가 들어오면 이동(Move) 상태로 전환한다.
- 이동(Move) 상태
    - 플레이어를 향해 직선으로 이동한다.
    - 위아래 방향은 무시하고 수평 이동만 처리한다.
    - 이동 속도는 에디터에서 조정할 수 있도록 노출한다.
    - 플레이어가 감지 범위를 벗어나면 대기(Idle) 상태로 복귀한다.
- 적 캐릭터 설정
    - 마우스 시점 방향으로 몸이 돌아가는 기능은 끈다.
    - 이동 방향으로 자동 회전하는 기능은 켠다.
- 상태가 바뀔 때마다 현재 상태 이름을 로그로 출력한다.
- 이 컴포넌트를 적 캐릭터(TPSEnemyCharacter)에 추가한다.

# 제한
- 공격/피격/사망 로직은 구현하지 않는다.
- 내비게이션 메시 기반 길 찾기는 사용하지 않는다.
- AI 컨트롤러는 사용하지 않는다.
