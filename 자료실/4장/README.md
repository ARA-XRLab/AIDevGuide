# 4장 자료 안내

이 폴더는 **4장 실습에 필요한 프롬프트, 실습 자료, 참고용 완성 프로젝트**를 모아둔 공간입니다.

4장에서는 Unreal Engine 기반의 TPS 프로젝트를 단계별로 구성합니다.  
독자는 `Prompt` 폴더의 프롬프트를 순서대로 활용하여 직접 실습을 진행할 수 있고,  
필요한 리소스는 `자료실` 폴더에서 확인할 수 있습니다.

또한 `TPSProject` 폴더에는 미리 완성된 참고용 프로젝트가 포함되어 있어,  
실습 결과를 비교하거나 전체 구조를 확인할 때 사용할 수 있습니다.

---

## 폴더 구성

- `Prompt/` : 4장 실습에서 사용할 단계별 프롬프트 파일
- `자료실/` : 책의 4장 실습에 필요한 추가 파일
- `TPSProject/` : 미리 완성해 둔 참고용 프로젝트
- `README.md` : 4장 자료 안내 문서

---

## 1. Prompt

`Prompt` 폴더에는 4장 실습에서 사용할 수 있는 단계별 프롬프트 파일이 들어 있습니다.

각 프롬프트는 책의 진행 순서에 맞게 번호가 붙어 있으며,  
AI 코딩 도구에 복사해서 입력할 수 있도록 작성되어 있습니다.

### 프롬프트 예시

- `[프롬프트 4.1-0 AGENTS.md 생성].md`
- `[프롬프트 4.1-1 AGENTS.md 프로젝트 지침 추가].md`
- `[프롬프트 4.1-2 프로젝트 초기 설정-소스 구조 정리].md`
- `[프롬프트 4.2-13 ATPSEnemyManager].md`

### 사용 방법

1. 책의 해당 실습 단계를 확인합니다.
2. `Prompt` 폴더에서 같은 번호의 프롬프트 파일을 찾습니다.
3. 프롬프트 내용을 AI 코딩 도구에 복사해서 입력합니다.
4. 생성되거나 수정된 코드를 확인합니다.
5. Unreal Editor에서 실행 결과를 확인합니다.

프롬프트는 실습을 더 쉽게 따라가기 위한 보조 자료입니다.  
실제 프로젝트에 적용한 뒤에는 반드시 Unreal Editor에서 동작을 확인해야 합니다.

---

## 2. 자료실

`자료실` 폴더에는 4장 책 내용에서 사용하는 추가 파일들이 들어 있습니다.

### 자료실 구성

- `Crosshair.png` : 사격 조준선 UI에 사용하는 이미지
- `Enemy.zip` : 적 캐릭터 실습에 필요한 리소스
- `Rifle.wav` : 총 발사 효과음
- `StarterContent.zip` : Unreal Engine 실습에 사용하는 Starter Content 자료

이 파일들은 책의 실습 과정에서 필요한 리소스입니다.  
본문에서 특정 파일을 사용하라고 안내하는 경우, 이 폴더에서 해당 파일을 찾아 사용하면 됩니다.

---

## 3. TPSProject

`TPSProject` 폴더는 4장 내용을 바탕으로 미리 만들어 둔 **참고용 완성 프로젝트**입니다.

### TPSProject 구성

- `TPSProject.uproject` : Unreal Engine 프로젝트 파일
- `Source/` : C++ 소스 코드
- `Content/` : 프로젝트 에셋
- `Config/` : 프로젝트 설정 파일
- `AGENTS.md` : AI 코딩 도구 작업 지침 파일

이 프로젝트는 독자가 직접 실습을 진행하기 전에 구조를 미리 확인하거나,  
실습 중 문제가 생겼을 때 결과를 비교하기 위한 참고 자료입니다.

따라서 실습을 처음부터 진행하려면 `Prompt` 폴더의 프롬프트를 순서대로 사용하는 것을 권장합니다.  
`TPSProject`는 완성 예제 또는 비교용 프로젝트로 활용하면 됩니다.

---

## 권장 사용 순서

1. 책의 4장 본문을 읽습니다.
2. `Prompt` 폴더에서 해당 단계의 프롬프트를 확인합니다.
3. AI 코딩 도구에 프롬프트를 입력합니다.
4. 생성된 코드와 프로젝트 변경 사항을 확인합니다.
5. 필요한 리소스는 `자료실` 폴더에서 가져옵니다.
6. 결과가 궁금하거나 비교가 필요하면 `TPSProject`를 참고합니다.

---

## 주의 사항

- `Prompt` 폴더의 파일은 AI 코딩 도구 입력용 예시입니다.
- `자료실` 폴더의 파일은 책의 실습 과정에서 사용하는 리소스입니다.
- `TPSProject`는 직접 따라 만드는 실습용 시작 프로젝트가 아니라, 미리 만들어 둔 참고용 프로젝트입니다.
- 실습 결과는 사용자의 Unreal Engine 환경이나 AI 코딩 도구의 응답에 따라 조금씩 달라질 수 있습니다.

---

## 4장 학습 흐름

4장은 다음 흐름으로 학습할 수 있도록 구성되어 있습니다.

1. 책 내용을 확인합니다.
2. 프롬프트를 확인합니다.
3. AI 코딩 도구에 입력합니다.
4. 코드 생성 및 수정 결과를 확인합니다.
5. 필요한 자료 파일을 적용합니다.
6. Unreal Editor에서 실행 결과를 확인합니다.
7. 참고용 프로젝트와 비교합니다.

이 자료를 활용하면 4장의 TPS 프로젝트 구현 과정을 단계별로 따라가면서,  
AI 코딩 도구를 이용한 Unreal Engine 프로젝트 제작 흐름을 함께 익힐 수 있습니다.
