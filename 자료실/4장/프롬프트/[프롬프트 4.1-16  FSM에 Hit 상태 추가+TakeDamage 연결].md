# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
적 AI에 피격 상태를 추가하고 데미지 수신과 연결한다.

# 요구사항
- FSM 컴포넌트에 NotifyHit() 기능을 추가한다.
  - 호출 즉시 피격(Hit) 상태로 전환하고 이동을 멈춘다.
  - 변수명 HitReactDuration 만큼 경직 시간을 설정한다. 에디터에서 조정할 수 있도록 노출한다.
  - 경직이 끝나면 플레이어가 유효할 경우, 이동(Move), 없으면 대기(Idle) 상태로 복귀한다.
- 피격(Hit) 상태는 매 프레임 처리 맨 앞에서 먼저 확인하며, 경직 중에는 이동/공격 상태로 전환되지 않는다.
- 적 캐릭터가 데미지를 받을 때 NotifyHit()를 호출한다.

# 제한
- 사망 로직 및 HP 처리는 구현하지 않는다.
- 피격 애니메이션은 구현하지 않는다.
- 기존 대기/이동/공격 로직은 필요한 부분만 최소한으로 수정한다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
