# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
적 사망 애니메이션과 땅속으로 가라앉아 사라지는 연출을 구현한다.

# 요구사항
## TPSEnemyAnimInstance
- 변수명 DeadMontage 추가, 에디터에서 AM_EnemyDead 연결 가능하게 노출
- PlayDeadAnim() 추가, DeadMontage 연결 시에만 재생

## TPSEnemyFSMComponent-NotifyDead()
- 사망 상태로 전환, 이동 즉시 멈추고 비활성화
- 몸통과 메시 충돌을 모두 끄고 PlayDeadAnim() 호출

## 가라앉기 연출
- sinkDelayTime(기본값 3.0f), sinkSpeed, removeZ를 에디터에서 조정 가능하게 노출
- 사망 후 sinkDelayTime 대기, 이후 매 프레임 아래로 이동, removeZ 이하 도달 시 오브젝트 제거

# 제한
- AM_EnemyDead 애셋은 에디터에서 직접 만든다.
- 기존 피격 처리 로직 및 중복 사망 방지 기능은 수정하지 않는다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
