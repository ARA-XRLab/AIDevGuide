# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
플레이어 오른손 소켓에 총기 메시를 부착한다.

# 요구사항
- 변수명 RifleMeshComp로 총기용 스켈레탈 메시 컴포넌트를 추가하고 에디터에서 확인할 수 있도록 노출한다.
- 총기는 플레이어 메시의 HandGrip_R 소켓에 부착하여 캐릭터와 함께 움직이도록 연결한다.
- 총기 모델은 코드에서 직접 불러온다.
- 로드 경로: /Game/Weapons/Rifle/Meshes/SKM_Rifle
- 총기는 연출용이므로 충돌을 비활성화한다.

# 제한
- 위치/회전 보정 변수는 추가하지 않는다.
- 기존 이동/카메라/사격/애니메이션 코드는 수정하지 않는다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
