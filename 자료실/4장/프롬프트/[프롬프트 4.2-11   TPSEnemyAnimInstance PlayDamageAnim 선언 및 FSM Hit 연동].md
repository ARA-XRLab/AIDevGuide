# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
적 캐릭터에 피격 애니메이션과 HP 기반 상태 분기를 추가한다.

# 요구사항
## TPSEnemyAnimInstance
- 변수명 DamageMontage를 추가하고 에디터에서 연결할 수 있도록 노출한다.
- PlayDamageAnim( ) 함수를 추가한다. DamageMontage가 연결된 경우에만 재생한다.

## HP 시스템
- 변수명 hp(기본값 3), maxHp(기본값 3)를 추가하고 에디터에서 조정할 수 있도록 노출한다.

## 피격 처리
- 데미지를 받으면 hp를 1 감소시킨다.
- hp 가 남아 있으면 피격(Hit) 상태로 전환하고 피격 애니메이션을 재생한다.
- hp 가 0 이하이면 사망(Dead) 상태로 전환한다.

# 제한
- 기존 FSM 상태 전이 구조는 변경하지 않는다.
- 애니메이션 애셋 생성 코드는 작성하지 않는다.
- 애니메이션 블루프린트 그래프 수정 코드는 작성하지 않는다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
