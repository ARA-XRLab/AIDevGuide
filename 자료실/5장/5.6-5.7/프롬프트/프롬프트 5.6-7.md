# Load 버튼 기능 구현 요청

- 누르면 파일 선택 창이 떠서 저장했던 JSON 파일을 고를 수 있어
- 파일을 불러오면 아래 내용이 전부 자동으로 적용돼야 해
  - Sprite Sheet Loader에 저장된 이미지가 다시 보여져
  - 이미지 위에 격자(그리드)도 다시 그려져
  - Frame Settings의 Rows, Columns, Total Frames, FPS, Loop 값이 전부 복원돼
  - Animation Preview에 첫 번째 프레임이 보여져
  - Play, Stop, Prev, Next 버튼이 바로 동작해
