# Save 버튼 기능 구현 요청

- 누르면 현재 설정된 모든 데이터를 파일로 저장해줘
- 저장할 데이터 목록
  * 현재 선택된 이미지가 뭔지 (이미지 경로)
  * Rows 값
  * Columns 값
  * Total Frames 값
  * FPS 값
  * Loop 체크 여부 (켜짐/꺼짐)
- 저장 파일 형식은 JSON으로 해줘
(나중에 언리얼에서 2D 게임 만들 때 이 데이터를 읽어서 쓸 예정이야)
- 파일 확장자는 .sprta 로 해줘 
- 저장할 때 파일 이름이랑 저장 위치를 직접 고를 수 있는 창이 떠야 해
- 저장 성공하면 화면에 "저장 완료!" 같은 메시지를 보여줘
