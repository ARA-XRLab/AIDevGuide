# ASpriteAnimActor 구현 요청

## 사전 조건
언리얼 에디터에서 "M_SpriteAnim_World" 머티리얼을 이미 만들어놨어.
이 머티리얼은 아래 3개의 파라미터를 가지고 있어.

- SpriteSheet: 스프라이트 시트 이미지 (TextureParameter)
- UVOffset: 현재 프레임의 시작 위치 (VectorParameter)
- UVSize: 현재 프레임의 크기 (VectorParameter)

## .sprta 파일 구조
@Saved\SpriteAnimationData.sprta
위 파일은 내가 만든 툴이 저장하는 .sprta 파일의 실제 내용이야.
이 구조에 맞춰서 읽을 수 있게 만들어줘.

## 구현 목표
"ASpriteAnimActor" 라는 액터 하나를 만들어줘.

## 액터 동작 방식
이 액터를 게임 월드에 놓으면

- Plane 메시 위에 스프라이트 애니메이션이 재생돼야 해
- 에디터에서 .sprta 파일 경로를 지정하면 게임이 시작될 때 자동으로 읽어서 애니메이션이 재생돼야 해
- FPS에 맞춰서 프레임이 자동으로 넘어가야 해
- 마지막 프레임이 끝났을 때 반복 재생 여부는 .sprta 파일 안에 있는 Loop 정보를 사용해줘
- 이미지는 .sprta 파일 안에 저장된 경로에서 불러와야 해

## 에디터에서 설정할 수 있어야 하는 것
- 상대경로를 이용해야해
- 어떤 .sprta 파일을 사용할지 
- 어떤 머티리얼을 사용할지

## 공통 조건
- 새 파일로 만들어줘 (기존 툴 플러그인 코드는 절대 건드리지 마)
- 플러그인 폴더가 아니라 Source/[내 프로젝트 이름] 폴더 안에 만들어줘
