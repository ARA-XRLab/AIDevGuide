# New 버튼 기능 구현 요청

- 누르면 모든 것을 처음 상태로 초기화해줘
- 초기화되는 항목
  - Sprite Sheet Loader → 이미지 사라지고 빈 흰색 박스로
  - Select Image 드롭다운 → "None"으로
  - Rows → 1
  - Columns → 1
  - Total Frames → 1
  - FPS → 12.0
  - Loop → 체크 켜짐
  - Animation Preview → 빈 흰색 박스로
  - Frame 표시 → "Frame: 0 / 1"로
- New 버튼 누르기 전에 "정말 초기화할까요?" 확인 창을 띄워줘
  (실수로 누르는 걸 방지하기 위해)
