# Select Image 미리보기 연동 구현 요청

Select Image를 누르면 이미지를 고를 수 있고,
선택한 이미지가 "Sprite Sheet Loader" 박스에 표시되도록 해줘

## "Sprite Sheet Loader" 박스
- "Sprite Sheet Loader" 박스 를 "미리보기 영역" 으로 말할게

## 이미지 선택 방식
- 에디터 표준 이미지 피커를 사용해줘
- 목록에서 이미지 썸네일이 텍스트 옆에 작게라도 보여야 해
- 이미지만 나와야 해

## 이미지 표시 방식
- 선택한 이미지가 미리보기 영역 보다 크면 → 이미지 비율 유지해서 미리보기 영역에 맞게 축소해서 보여줘
- 선택한 이미지가 미리보기 영역 보다 작으면 → 이미지 비율 유지해서 미리보기 영역에 맞게 확대해서 보여줘
- 선택한 이미지는 미리보기 영역 한가운데에 보여줘

## 반드시 지켜야 하는 동작 조건
- 아래 조건들은 AI가 실수를 많이 하는 부분이니 특히 신경써줘
- 선택한 이미지를 고르자마자 미리보기 영역에 바로 나와야 해
  두 번 클릭하거나 다른 곳을 눌러야 나오면 안 돼
- 선택한 이미지는 처음부터 비율을 유지하고 미리보기 영역에 꽉 차게 나와야해
- 표준 위젯 조합으로 우회하지 말고, 필요하면 커스텀 위젯으로 직접 그려줘
