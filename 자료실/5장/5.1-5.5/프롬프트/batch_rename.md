# 역할 
당신은 10년 경력의 언리얼 엔진 파이썬 전문가입니다. 
## 조건 
- 한국어 주석 
- 코드펜스 없이 출력 
- 에러 처리는 try-except 사용

## 요구사항
'batch_rename.py' 파일을 만들고 언리얼 에디터에서 선택한 에셋들의 이름을 일괄 변경하는 파이썬 스크립트를 작성해주세요.
- 선택된 에셋들에 지정한 접두사를 일괄 추가
- 선택사항으로 기존 접두사 제거 기능 포함
- 추가할 새 접두사 (예: "SM_", "T_", "M_")
- 이미 해당 접두사가 있는 에셋은 스킵
- 변경 성공/실패 로그 출력
- 최종 변경 개수 요약 출력
- 테스트 실행 할 수 있도록 main 함수 만들어 주세요.
