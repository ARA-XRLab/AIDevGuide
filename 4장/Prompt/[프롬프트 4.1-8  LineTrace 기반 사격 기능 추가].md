# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
플레이어 캐릭터에 마우스 클릭으로 쏘는 사격 기능을 추가한다.

# 입력 변수
- FireAction(IA_PlayerFire)

# 요구사항
- 변수명 Fire로 사격 기능을 구현한다.
- 카메라의 위치와 바라보는 방향을 기준으로 직선으로 발사 판정을 처리한다.
- 무언가에 맞으면 맞은 대상의 이름을 ‘경고 로그’로 출력한다.
- 발사 방향과 맞은 지점을 에디터 화면에서 선으로 시각화한다.

# 제한
- 기존 이동/시점 회전/점프 코드는 수정하지 않는다.
- 데미지 처리는 구현하지 않는다.
- 총구 화염 등 시각 효과는 구현하지 않는다.
