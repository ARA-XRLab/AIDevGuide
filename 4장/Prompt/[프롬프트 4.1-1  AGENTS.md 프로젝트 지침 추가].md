
# 역할
Unreal Engine 5 C++ 개발자. 기존 AGENTS.md 규칙을 따른다.

# 작업
기존 AGENTS.md에 이 프로젝트 전용 개발 지침을 추가한다.

# 추가할 내용
## 프로젝트 철학
- 기능 구현-동작 확인-프롬프트 개선-반복

## 개발 단계
- 프로토타입: 핵심 기능, 단순 구조, 빠른 실험
- 알파: 구조 안정화, 리팩터링, 시스템 분리
- 베타: 최적화, 폴리싱

## 협업 원칙
- 초급~중급 개발자도 따라가기 쉽게 작성
- 책임 분리 명확화
- 프롬프트로 재현 가능한 구조 지향

## 코딩 규칙
- 클래스, 변수, 함수마다 한글 주석 작성
- Tick은 기본 비활성화, 필요 시만 활성화
- 디버그 로그는 기본 노란색, 오류/위험은 빨간색
- 에디터 조정 값은 Category 지정 후 노출

## 소스 구조
- 헤더는 Public
- 구현은 Private

## 에디터 연동
- 블루프린트 애셋은 에디터에서 직접 생성
- 애셋 경로를 코드에 직접 적지 않음
- 수치는 에디터에서 조정 가능하게 노출

## 응답 형식
- 새 파일: 전체 코드 출력
- 기존 파일 수정: 바뀐 부분만 출력

# 제한
- 기존 AGENTS.md 내용 삭제/변경 금지
- C++ 코드 작성 금지
