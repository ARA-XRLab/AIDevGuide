# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
플레이어 캐릭터에 3인칭 시점 카메라를 추가한다.

# 요구사항
- 변수명 SpringArm으로 스프링 암을 추가한다. 마우스 시점 방향을 따라 회전한다.
- 카메라의 거리는 에디터에서 직접 조정할 수 있도록 노출한다.
- 변수명 Camera 로 카메라를 추가하고 SpringArm 끝에 붙인다. 카메라 자체는 별도로 회전하지 않는다.
- 캐릭터 몸통은 카메라가 바라보는 좌우 방향을 따라 회전한다.
- 변수명 StaticMesh로 임시 외형 메시를 추가한다. 에디터에서 직접 연결할 수 있도록 한다.

# 제한
- 이동 및 회전 입력은 지금 추가하지 않는다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
