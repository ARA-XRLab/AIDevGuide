# 역할
언리얼 엔진 5 C+ + 개발자, AGENTS.md 규칙을 따른다.

# 작업
플레이어 캐릭터의 임시 외형을 스켈레탈 메시로 교체한다.

# 요구사항
- 기존 StaticMesh 컴포넌트를 헤더와 생성자에서 완전히 제거한다.
- 코드에서 SKM_Manny_Simple를 직접 불러와 기본 메시 슬롯에 적용한다.
  - 로드 경로: /Game/Characters/Mannequins/Meshes/SKM_Manny_Simple
- 메시의 위치와 회전 보정 값은 에디터에서 조정할 수 있도록 노출한다.
- 플레이어 몸통(캡슐)은 사격 판정에서 제외한다.
- 플레이어 메시는 물리 충돌 없이 사격 판정만 받도록 설정한다.
- 오브젝트 타입은 Pawn으로 유지한다.
- 사격 판정은 메시 기준으로 처리한다.

# 제한
- 이동/점프/카메라/사격 로직은 수정하지 않는다.
- 애니메이션 블루프린트는 연결하지 않는다.
- 블루프린트 생성 코드는 작성하지 않는다.

# 성공 조건
- 플레이어가 SKM_Manny_Simple로 표시된다.
- 사격 판정이 메시 기준으로 동작한다.
- 기존 이동/점프/카메라가 정상적으로 유지된다.
