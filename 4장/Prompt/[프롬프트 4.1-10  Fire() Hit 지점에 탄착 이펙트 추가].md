# 역할
언리얼 엔진 5 C+ + 개발자, AGENTS.md 규칙을 따른다.

# 작업
사격 시 총알이 맞은 지점에 이펙트를 생성한다.

# 요구사항
- 변수명 BulletImpactFX 로 이펙트를 추가하고, 에디터에서 P_BulletEffect를 연결할 수 있도록 노출한다.
- 맞은 지점에 이펙트를 1회 생성하고, 맞은 표면의 법선 방향으로 회전한다.
- 표면에 파묻히지 않도록 살짝 띄워서 생성하며, 띄움 거리는 에디터에서 조정할 수 있도록 노출한다.
- 이펙트가 연결되지 않은 경우, 조용히 건너뛴다.

# 제한
- 기존 발사 판정/입력/크로스헤어 코드는 수정하지 않는다.
