# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
적 전용 애니메이션 인스턴스 클래스를 만들고 FSM 상태를 애니메이션 블루프린트로 전달한다.

# 요구사항
- 클래스 이름: TPSEnemyAnimInstance
- 언리얼 엔진 기본 애니메이션 인스턴스를 상속받아 만든다.
- 현재 FSM 상태를 나타내는 변수 AnimState를 추가하고 에디터에서 확인할 수 있도록 노출한다. 기본값은 Idle이다.
- FSM 컴포넌트를 다음과 같이 수정한다.
  - 내부에서만 사용하는 TPSEnemyAnimInstance 참조 변수 Anim을 추가한다.
  - 게임 시작 시 메시의 애니메이션 인스턴스를 Anim에 저장한다.
  - 저장 성공 시 AnimState를 Idle로 초기화한다.
  - 상태가 바뀔 때마다 AnimState도 함께 동기화한다.
- 적 캐릭터 생성자에서 ABP_TPSEnemy를 코드로 불러와 메시에 연결한다.
  - 로드 경로: /Game/Animation/ABP_TPSEnemy

# 제한
- 애니메이션 그래프 관련 코드는 작성하지 않는다.
- Idle 외 다른 애니메이션 상태 로직은 최소한으로만 구현한다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
