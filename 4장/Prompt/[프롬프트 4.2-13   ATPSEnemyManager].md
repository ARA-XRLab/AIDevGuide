# 역할
언리얼 엔진 5 C++ 개발자. AGENTS.md 규칙을 따른다.

# 작업
적을 주기적으로 자동 생성하는 ATPSEnemyManager 클래스를 구현한다.

# 요구사항
- 언리얼 엔진 기본 액터를 상속받아 만든다. 매 프레임 업데이트는 사용하지 않는다.
- 에디터에서 조정 가능한 변수
  - 생성할 적 클래스 타입
  - 생성 주기(float)
  - 최대 동시 존재 수(int32)
  - 스폰 위치를 찾기 위한 태그 이름
- 게임 시작 시 태그로 스폰 위치 액터를 자동으로 수집하고, 조건이 충족되면 타이머를 시작한다.
- 타이머마다 현재 적 수가 최대 수 미만일 때 랜덤 스폰 위치에서 적을 생성한다.

# 제한
- 스폰 위치 액터 배치는 에디터에서 직접 진행한다.
- 길 찾기 및 AI 컨트롤러 관련 설정은 추가하지 않는다.
