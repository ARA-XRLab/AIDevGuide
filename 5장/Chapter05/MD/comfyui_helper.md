# 요구사항
ComfyUI 통신 헬퍼 클래스 작성 (표준 라이브러리만 사용, 외부 패키지 금지)
사용자가 전송한 워크플로우를 이용해 이미지를 생성합니다.

## 작업 목표
`comfyui_helper.py` 파일에 `ComfyUIHelper` 클래스를 작성해 주세요.

## 메서드
1. 메서드의 이름은 **generate_image** 입니다.
2. 워크플로우 전송
3. 완료될 때까지 대기 (너무 오래 걸리면 에러)
4. 결과 이미지 다운로드 및 저장

## 조건
- 한국어 주석
- 코드펜스 없이 클래스만 출력
- 워크플로우: 스크립트와 같은위치의 `workflows/texture_basic.json`
- 출력 폴더: 스크립트와 같은위치의 `output/`
