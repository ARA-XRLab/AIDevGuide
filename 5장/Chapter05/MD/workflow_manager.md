# 요구사항
ComfyUI 워크플로우 템플릿 관리 클래스 작성 (표준 라이브러리만 사용, 외부 패키지 금지)
템플릿 디렉토리에서 모든 JSON 파일을 자동으로 불러오며, 디렉토리가 없으면 새로 생성합니다.

## 작업 목표
`workflow_manager.py` 파일에 `WorkflowManager` 클래스를 작성해 주세요.

## 메서드
1. 메서드의 이름은 **get_workflow** 입니다.
   - 템플릿 이름과 프롬프트·시드·이미지 크기를 받아 완성된 워크플로우를 반환합니다
   - 원본 템플릿은 수정하지 않습니다
   - 긍정 프롬프트, 부정 프롬프트, 시드, 이미지 크기를 동적으로 교체합니다

## 조건
- 한국어 주석
- 코드펜스 없이 클래스만 출력
