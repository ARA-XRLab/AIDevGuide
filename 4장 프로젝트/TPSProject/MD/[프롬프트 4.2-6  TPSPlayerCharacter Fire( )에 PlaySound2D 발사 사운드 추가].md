# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
플레이어 사격 시 총소리를 재생한다.

# 요구사항
- 변수명 BulletSound로 사운드 변수를 추가하고 에디터에서 사운드 애셋을 연결할 수 있도록 노출한다.
- 사격 기능에서
  - 기존 애니메이션 호출 및 발사 판정은 그대로 유지한다.
  - 애니메이션 재생 직후 BulletSound를 2D 사운드로 재생한다.
  - BulletSound 가 연결된 경우에만 재생한다.
- 사운드 재생에 필요한 헤더를 추가한다.

# 제한
- 기존 이동/카메라/애니메이션/발사 판정 코드는 수정하지 않는다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
