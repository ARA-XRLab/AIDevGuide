# 역할
언리얼 엔진 5 C++ 개발자, AGENTS.md 규칙을 따른다.

# 작업
적 AI에 공격 상태를 추가한다.
# 요구사항
- 이동(Move) 상태에서 플레이어와의 거리가 공격 범위 이내이고 쿨다운이 없을 때 공격(Attack) 상태로 전
환한다.
- 공격 범위는 에디터에서 조정할 수 있도록 노출한다.
- 공격(Attack) 상태
  - 진입 즉시 이동을 멈춘다.
  - 변수명 AttackHitRadius 반경 안의 캐릭터를 감지한다.
  - 감지된 대상에게 변수명 AttackDamage 만큼 데미지를 전달한다.
  - 공격 후 변수명 AttackCooldown 만큼 대기 시간을 설정하고 이동(Move) 상태로 복귀한다.
  - 감지된 대상이 없으면 대기(Idle) 상태로 복귀한다.
- 쿨다운은 매 프레임마다 줄어들며, 쿨다운이 남아 있으면 공격 상태로 다시 진입하지 않는다.

# 제한
- 피격/사망 로직은 구현하지 않는다.
- 공격 애니메이션은 구현하지 않는다.
- 기존 대기/이동 로직은 필요한 부분만 최소한으로 수정한다.
- 변수명은 위에 지정한 이름 그대로 사용한다.
